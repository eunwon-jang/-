# quarter_car 서스펜션 시스템 분석

(quarter-car) 서스펜션 시스템의 모델링, 시뮬레이션 및 주파수 응답을 분석하고 승차감에 가장 큰 영향을 주는 변수를 분석하여 파라미터를 최적화한다.

## 📌 포함 내용

- 시스템 모델링 및 전달함수 도출
- MATLAB 기반 시뮬레이션 (단계 응답, 사인파, 요철 입력)
- 주파수 응답 (Bode Plot), RMS 가속도 계산
- 결과 보고서 (PDF 포함)
